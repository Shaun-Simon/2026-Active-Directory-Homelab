# 2026 Active Directory Homelab Project

## 📌 Overview
This is a fully virtualised Active Directory Homelab Project that is built within my Proxmox Homelab to simulate a real world enterprise style IT enviornment. Designed to replicate common infrastructure, administration, and support tasks peformed in daily operations.

This lab will includes a Windows Server Domain Controller, multiple domain joined client machines, organisational units, group policies, and network file shares.

-----

## 🎯 Purpose
The purpose of this project is to:

- Develop hands-on experience with Active Directory environments
- Simulate real-world IT enviornment
- Build practical troubleshooting skills
- Create a professional portfolio project

-----

## 🧠 Skills Demonstrated

- Active Directory Domain Services (AD DS)
- User & Group Management
- Group Policy (GPO) Configuration
- Windows Server Administration
- Network Configuration (DNS, DHCP concepts)
- Troubleshooting (login issues, permissions, connectivity)
- Virtualisation using Proxmox
- Documentation & Technical Reporting

-----

## 🛠️ Technology Used

- Proxmox VE (Virtualisation Platform)
- Windows Server (Domain Controller)
- Windows 10/11 (Client Machines)
- Active Directory Domain Services
- Group Policy Management
- SMB File Shares
